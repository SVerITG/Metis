# Why R for Epidemiology

## Learning objectives
- Explain why reproducibility matters in epidemiologic analysis.
- Describe practical advantages of R over spreadsheet-only workflows.
- Recognize when code improves quality control and collaboration.

## Prerequisites
- Basic epidemiology and tabular data experience.

## Content
R matters in epidemiology because analytic decisions need to be transparent, repeatable, and easy to update. Spreadsheet workflows often hide transformations, make version drift hard to detect, and encourage manual copy-paste errors. R turns those steps into visible code.

That visibility matters for surveillance summaries, manuscript tables, reproducible figures, and rerunning analyses after data updates. The goal is not coding for its own sake. The goal is making analytic work easier to inspect, revise, and trust.

## Key takeaways
- Code improves transparency and repeatability.
- R reduces manual copy-paste and undocumented transformations.
- Reproducibility is a quality issue, not just a technical preference.

## Self-check questions
1. Why is reproducibility important in epidemiology?
2. What is one common spreadsheet risk?
3. How can R help when data are updated?
4. Why is visible code useful in collaboration?
5. What is the main reason to learn R here?

## Answer key
1. Because analyses often need verification, updating, and auditability.
2. Hidden manual edits or copy-paste errors.
3. The same script can rerun the workflow consistently.
4. Others can inspect and reproduce the analytic steps.
5. To produce more reliable and maintainable analytical work.
