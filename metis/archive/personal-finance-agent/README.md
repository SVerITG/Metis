# Personal Finance

Personal Finance is the financial awareness and market trend monitoring agent for the second-brain system.

Its job is to:

- monitor market trends relevant to the user's goals
- filter financial developments for project relevance
- produce concise financial briefs
- connect economic context to active projects

Invocation: `/personal-finance`

Personal Finance works under Metis. Has internet permission for public market data (like News Radar has for news).

## Files

- `contract.md`: scope, authority, and limits
- `system-prompt.md`: concise future implementation prompt
