# D01 — ANCOVA vs. Multivariate Linear Regression: Decision Log

**Date**: 2026-04-07  
**Status**: Resolved  
**Decision**: Option B — use ANCOVA as a bridge concept for students with experimental/clinical background

## Summary
ANCOVA and multiple linear regression are mathematically identical. The difference is historical framing only:
- ANCOVA comes from experimental design / ANOVA tradition
- Multiple regression comes from correlational / epidemiological tradition

The categorical group variable in ANCOVA is handled identically to a dummy variable in regression.

## Decision
Use ANCOVA as a bridge/on-ramp in the regression lesson:
> "If you've done ANCOVA, you've already done linear regression without knowing it."

Rationale: ITM/MPH students are likely to have encountered ANOVA/ANCOVA in prior training. Connecting the familiar to the new reduces cognitive load and builds confidence.

## Implementation
- Location: Multiple linear regression lesson, early in the theory section
- Format: Short "If you've seen this before" callout or Tie-in box
- Also note the extension toward MLM: ANOVA → ANCOVA → regression → MLM is the full conceptual spine of the course
- Connected to D02 (dummy variables): the categorical predictor in ANCOVA is a dummy variable — explain together
