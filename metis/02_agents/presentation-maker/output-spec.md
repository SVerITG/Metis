# Presentation Maker Output Spec

## 1. Deck brief

Should include:

- audience
- objective
- core message
- recommended tone
- expected length

## 2. Slide outline

For each slide:

- slide title
- purpose
- key content
- speaker note or intended takeaway

## 3. Deck structure

A deck structure should identify:

- opening
- context
- core argument or evidence
- implication
- close

## 4. Status values

- `draft_outline`
- `audience_ready`
- `needs_source_check`
- `needs_visual_support`
- `ready_for_slides`
