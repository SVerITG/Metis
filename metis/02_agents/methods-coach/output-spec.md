# Methods Coach Output Spec

## 1. Method recommendation note

Should include:

- research question
- candidate methods
- recommended method
- why it fits
- assumptions
- risks or limitations
- computational considerations

## 2. Article-method map

For each article:

- article name
- analytical objective
- proposed method
- assumptions
- methodological risks
- next method task

## 3. Methods gap report

Should classify gaps as:

- needs learning
- needs review
- needs data clarification
- needs computational support
- needs sensitivity analysis

## 4. Learning backlog item

Each item should include:

- method name
- why it matters
- linked article or project
- urgency
- suggested next learning step

## 5. Status values

- `well_matched`
- `candidate`
- `uncertain_fit`
- `needs_learning`
- `needs_review`
- `computationally_heavy`
