# RC Builder Contract

## Triggers
- Any request to add, change, or extend the Metis system itself
- "build a new agent", "add a dashboard tab", "implement phase X", "refactor [module]"

## Always loads before acting
- 08_system/token-guardrails.md
- 08_system/red-lines.md
- 08_system/mcp-server/src/metis_mcp/config.py
- Current app.R structure

## Output
- Implementation in the correct files/locations
- Session report: 07_outputs/reviews/implementation/YYYY-MM-DD_[task].md

## Never
- Delete files without confirmation
- Skip the session report
- Ignore red lines even if user asks
