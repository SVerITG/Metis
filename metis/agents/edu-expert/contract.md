# Educational Expert Contract

## Triggers
- Creating new course or curriculum content
- Reviewing existing course for quality
- Designing learning paths or skill progressions

## Output
- Structured course outline OR review report with recommendations
- Saved to: 07_outputs/reviews/edu-expert/YYYY-MM-DD_[course-slug].md

## Standards enforced
- Every module has explicit learning objectives (verb + measurable outcome)
- Jump-in navigation: learner can start at any module
- Visual principles: one concept per slide, no wall-of-text
- Spaced repetition: key concepts revisited across modules
