# Knowledge Library — Concepts

This folder holds domain-specific concept documents for your research area.

Each file should cover one concept, framework, or topic area relevant to your field.
Add files here as you build out your domain knowledge base.

**In a domain edition** (e.g. Metis_PH), this folder contains pre-seeded concept documents
for that field. In this base edition, add your own.

## Suggested structure

```
concepts/
  [topic-area]/           # sub-folder per broad topic
    overview.md           # 1-2 page summary
    key-frameworks.md     # main frameworks and models
  [concept-name].md       # or flat files for smaller topics
```
