# R for Epidemiologists

This course introduces practical R workflows for epidemiologists and public health researchers. It focuses on data import, cleaning, transformation, visualization, reproducible analysis, and common epidemiologic tasks rather than on computer-science abstraction.

## Audience
- Epidemiologists new to R
- Public health analysts moving from spreadsheets to code
- Researchers building reproducible data workflows

## Structure
- 12 lessons
- Starts with setup, objects, and importing data
- Moves through wrangling, visualization, and epidemiologic summaries
- Ends with modeling, reproducibility, and communicating outputs

## Core resources
- R for Data Science
- Posit documentation
- Epidemiology-focused R tutorials and open materials
- Related Metis cards in `knowledge/library/methods/` and `knowledge/library/concepts/`
