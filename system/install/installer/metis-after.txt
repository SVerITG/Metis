  METIS IS INSTALLED — YOUR NEXT STEPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


  ① INSTALL CLAUDE DESKTOP  (free — required)
  ─────────────────────────────────────────────
  Metis connects to Claude Desktop automatically on first open.

  Download from:   https://claude.ai/download

  Once installed, open Claude Desktop — Metis will register itself
  and your setup wizard will start on the first conversation.


  ② START THE METIS DASHBOARD  (if you chose Full install)
  ──────────────────────────────────────────────────────────
  Double-click "Metis" on your desktop, or start it manually:

    Documents\Metis\system\install\windows\run-dashboard.bat

  Then open in your browser:   http://localhost:8080

  Your setup wizard is also available at:   http://localhost:8080/setup


  ③ COMPLETE YOUR SETUP WIZARD
  ─────────────────────────────
  The wizard personalises Metis to your research domain, your
  working style, and your projects. It takes about 3 minutes.

  • In Claude Desktop:  it starts automatically on first open
  • In the dashboard:   go to http://localhost:8080/setup
  • Any time later:     type  /metis_config  in Claude


  ④ LEARN CLAUDE DESKTOP AND CLAUDE CODE
  ────────────────────────────────────────
  Claude Desktop — conversational AI for research (no coding):
    https://claude.ai/download

  Claude Code — for developers and power users:
    https://docs.anthropic.com/en/docs/claude-code/getting-started

  Metis works with both. Claude Desktop is the natural starting point.

  YOUR ANTHROPIC API KEY — FULL GUIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Metis uses the Anthropic API to power its 34 specialist agents
  (Epidemiologist, Writing Partner, Librarian, Methods Coach, and more).
  You need a free API key from Anthropic to use them.

  HOW TO GET YOUR FREE API KEY
    1. Go to:  https://console.anthropic.com
    2. Sign up or log in (free — no credit card needed to start)
    3. Click "API Keys" in the left sidebar, then "Create Key"
    4. Copy the key (it looks like  sk-ant-api03-…)

  WHERE METIS STORES IT
    Locally only, in:   Documents\Metis\system\.env
    It is never uploaded or shared — it is sent only to Anthropic's own
    API when you run an agent. Paste or update it during the installer
    wizard, in the dashboard setup, or by editing that .env file.

  COSTS
    Most daily research tasks cost a few cents. Metis shows you exactly
    what each agent run costs — there are no hidden charges. You can set a
    monthly spending limit in the Anthropic console.
