# Specialist intros — for Metis routing announcements

One-line role description per specialist, used in the `Handing this to [Agent] — [one-line role]` form defined in `metis-persona.md`. Kept neutral and terse — no invented personalities, no gendered pronouns.

Use the role line verbatim. Adjust only the complexity descriptor and the "what they'll do" tail.

## Knowledge & literature

- **Librarian** — literature and source specialist.
- **Content Harvester** — extraction from web, PDFs, YouTube, GitHub.
- **News Radar** — world and research news monitor.
- **News Aggregator** — RSS curation and signal tagging.

## Research methods & writing

- **Epidemiologist** — study design and methodology review.
- **Methods Coach** — statistics, sampling, R methodology.
- **PhD Architect** — thesis structure and article alignment.
- **Writing Partner** — prose, argument flow, manuscript editing.
- **Presentation Maker** — slide decks, briefing decks, visual summaries.
- **Visualization Maker** — diagrams, charts, figures.

## Engineering

- **DHIS2 Expert** — DHIS2 platform specialist: server, metadata, tracker programs, dashboards, app development, implementation strategy.
- **Software Engineer** — code, debugging, scripting.
- **Frontend Designer Builder** — UI builds, design systems, web interfaces.
- **Design Auditor** — UI critique and prioritized improvements.
- **Builder** — new apps, tools, MCP servers.
- **RC Builder** — internal Metis extensions: agents, dashboard phases, MCP tools.

## Data

- **Data Analyst** — dataset profiling, cleaning, comparison.
- **Data Guardian** — PII detection and transmission control.
- **Cybersecurity** — URL validation, prompt-injection defense, agent audit.

## Learning & career

- **Learning Architect** — curriculum design, competency maps, spaced repetition.
- **Learning Coach** — skill progression and study planning.
- **Course Builder** — end-to-end course construction.
- **Edu Expert** — instructional design and content review.
- **Career Coach** — CV, applications, EU career strategy.

## Ops

- **Meeting Memory** — transcription, structuring, briefing notes.

## Usage examples

> "Handing this to Librarian — literature specialist. Deep pass: citation network for the spatial-clustering paper. Output to `outputs/reviews/librarian/`."

> "Two passes. Epidemiologist — study design review on the case-control draft. Then Writing Partner — methods section prose. One output file each."

> "Quick one — Data Guardian first to confirm the file is safe to share, then Methods Coach for the variance question."

## When no match exists

If a request fits no specialist, route to **HR Talent Spotter** to assess whether a new agent is needed. Phrase: "No specialist covers this. Handing to HR Talent Spotter to scope the gap."
