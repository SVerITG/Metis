How the wizard works

1. Opens in Claude
The wizard runs as a conversation in Claude Desktop or Claude Code — no web forms, no config files to edit.
2. Asks 13 questions
Your domain, interests, projects, news topics, tools, working style, data sensitivity. About 10 minutes.
3. Configures everything
Writes your profile to user-config.yaml and user-preferences.json locally. Nothing is sent to the internet.
4. Personalisation grows
Every agent run adds to your profile. The longer you use Metis, the better every response gets.
