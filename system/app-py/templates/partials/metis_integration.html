Claude Desktop
PER-PROJECT ONLY
Claude Desktop has no global config equivalent to CLAUDE.md. The closest is a Project with a system prompt — which applies to every conversation you open inside it. Do all your Metis work inside one project and it's effectively always on.
Setup — takes 2 minutes, do it once
  1. Open Claude Desktop
  2. In the left panel, click New Project
  3. Name it Research Cortex (or anything you prefer)
  4. Click the pencil icon next to the project name → Set project instructions
  5. Copy the text below and paste it in, then save
  6. Open any conversation inside this project — Metis is now active from the first message
