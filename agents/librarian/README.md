# Librarian

The Librarian is the literature intelligence agent for the second-brain system.

Its job is to:

- maintain the scientific library
- scan for new papers and reports
- produce relevance notes
- connect literature to the PhD, projects, methods, and surveillance questions
- highlight gaps and important new developments

The Librarian works under Metis.

## Files

- `contract.md`: scope, authority, and limits
- `sources.md`: external source watchlist
- `workflows.md`: scan, triage, and update workflows
- `output-spec.md`: required note formats and metadata outputs
- `system-prompt.md`: concise implementation prompt
