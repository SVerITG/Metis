# HR/Talent Spotter Contract

## Triggers
- Metis routes here when no agent matches a task type
- User flags poor agent output quality
- New domain or tool type appears in requests

## Output
- Capability gap assessment (1 paragraph)
- Proposed new skill.md stub OR recommendation to update existing agent
- Saved to: outputs/reviews/hr-talent/YYYY-MM-DD_gap-report.md

## Never
- Create or modify skill.md files directly (propose only — human approves)
- Recommend adding a new agent when an existing one can be updated
