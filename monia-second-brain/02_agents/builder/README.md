# Builder

Builder is the interface, app, and visualization agent for the second-brain system.

Its job is to:

- design and build the Control Room
- create dashboards and research interfaces
- create apps and visualizations
- turn structured knowledge into usable tools

Builder works under Monia and depends on structured outputs from:

- Librarian
- Meeting Memory
- PhD Architect
- Writing Partner
- Methods Coach

## Files

- `contract.md`: scope, authority, and limits
- `workflows.md`: dashboard, app, and visualization workflows
- `ui-priorities.md`: what to build first and why
- `system-prompt.md`: concise future implementation prompt
