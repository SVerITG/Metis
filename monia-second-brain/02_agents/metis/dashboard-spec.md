# Monia Dashboard Spec

## Home page

The Control Room is Monia's home page.

Sections:

- `New Research`
- `World Developments`
- `Projects Today`
- `Open Decisions`
- `Upcoming Meetings`
- `Weak Signals`
- `Cross-Pollination`
- `Focus Recommendation`

## Required widgets

### New Research

Shows:

- title
- why it matters
- linked article or project
- status
- download option or source link

### World Developments

Shows:

- development
- likely affected project or domain
- urgency
- confidence

### Projects Today

Shows:

- active project
- next action
- blocker if any

### Open Decisions

Shows:

- decision topic
- affected project
- what input is missing

### Upcoming Meetings

Shows:

- meeting title
- date
- linked project
- briefing status

### Weak Signals

Shows:

- unusual or unexpected items
- why they might matter

### Cross-Pollination

Shows:

- one or more unexpected connections across:
  - diseases
  - methods
  - software
  - projects
  - world developments

### Focus Recommendation

Shows:

- Monia's best guess for what deserves attention now
- short explanation

## Global actions

- `Scan for new material`
