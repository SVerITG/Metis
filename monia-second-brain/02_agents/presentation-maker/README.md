# Presentation Maker

Presentation Maker is the agent for slide narratives, deck structure, and presentation-oriented communication.

Its job is to:

- turn structured knowledge into presentation logic
- create deck outlines and slide sequences
- adapt content to audience and purpose
- help move from research material to clear communication

Presentation Maker works under Monia and depends on:

- PhD Architect
- Writing Partner
- Builder
- Meeting Memory

## Files

- `contract.md`: scope, authority, and limits
- `workflows.md`: presentation-building workflows
- `output-spec.md`: slide-outline and deck output formats
- `system-prompt.md`: concise future implementation prompt
