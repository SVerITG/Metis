# Multilevel Analysis and Spatial Scan Statistics Course

Teaching and learning material for multilevel modelling (MLM) and supercomputer-based spatial scan statistics. Includes a Shiny app for course delivery and example R scripts.

## What it does
- Course material for multilevel analysis (logistic, ordinal, Poisson/NB, three-level, longitudinal, GEE)
- Example R scripts demonstrating each model type
- `mlm-app` — Shiny application for course delivery
- LEMMA and other MLM resources

## Key content
### Example R scripts
- `01_logistic_multilevel.R`
- `01_multilevel_analysis_general.R`
- `01_teaching_intro_MLM.R`
- `02_ordinal_poisson_nb_models.R`
- `02_three_level_models.R`
- `02_visualizing_random_effects.R`
- `03_centering_and_crosslevel.R`
- `03_longitudinal_binary_outcomes.R`
- `03_longitudinal_growth_models.R`
